A professional, responsive resume website built with HTML, CSS, and JavaScript.

✨ **Professional Design** - Clean, modern layout optimized for readability  
📱 **Responsive** - Looks great on desktop, tablet, and mobile devices

## Getting Started

### Option 1: View Locally
1. Clone the repository:
   ```bash
   git clone https://github.com/noahlwayne-ui/resume-website-.git
   ```

2. Navigate to the project directory:
   ```bash
   cd resume-website-
   ```

3. Open `index.html` in your web browser

### Option 2: Deploy to GitHub Pages
1. Go to your repository settings
2. Scroll to "GitHub Pages" section
3. Select "main" branch as source
4. Your site will be available at `https://noahlwayne-ui.github.io/resume-website-/`

## Customization

### Update Contact Information
Edit the contact links in `index.html`:
```html
<span>📧 noahlwayne@gmail.com</span>
<span>🔗 <a href="https://linkedin.com/in/yourprofile">LinkedIn</a></span>
```

### Modify Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #1a3a52;
    --secondary-color: #2c5aa0;
    --accent-color: #e8b74a;
}
```

### Add New Sections
Add new sections using the same structure as existing ones:
```html
<section class="section">
    <h2>NEW SECTION</h2>
    <!-- Your content here -->
</section>
```

## Printing

To print your resume as PDF:
1. Open the website in your browser
2. Press `Ctrl+P` (Windows) or `Cmd+P` (Mac)
3. Select "Save as PDF" as your printer
4. Adjust margins if needed
5. Download the PDF
